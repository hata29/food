# 📅 週次メニュー

左のサイドバーから週ごとのページへ。1ページに **献立 + 仕込み + 買い物リスト + 翌週繰越 + 振り返り** をまとめる。

## 命名規則

- `YYYY-Www.md`（例: `2026-W18.md` = 2026年第18週）

## ページ構成

| セクション | 内容 |
|---|---|
| 献立 | 平日・休日の各日メニュー |
| 仕込み | 日曜30分で来週の時短になる仕込み |
| 買い物リスト | カテゴリ別 + 「※家にあるか確認」 |
| 翌週繰越候補 | 使い切れない見込みの食材（次週で優先消費） |
| メモ・振り返り | 実食後の気づき・買い物のミス・申し送り |

## 組み方の方針

CLAUDE.md「献立作成時」セクション参照。要点：

- 食材使い切り最優先
- 平日夜2〜3食 + 平日昼2〜3食（在宅日のみ、10分以内）
- 木金夜は外食前提でOK
- 連続するタンパク源は3日連続避ける程度（厳しくはしない）
- 日曜30分仕込みで来週の時短に効くものを提案
