# 🍜 外食リスト

行きつけ・お気に入りの店を `restaurants/{ジャンル}/{店名}.md` で管理。

## 運用ルール

- ジャンルディレクトリは初出時に作る（先に空フォルダは作らない）
- 訪問履歴・シーン・学びを**累積式**で追記（過去の記録は消さない）
- シーンは複数可：一人飲み / ご褒美 / 仕事後 / デート など、後から絞り込みに使う
- 完璧を目指さない、3行でも記録する

## ハイブリッド提案

「今夜どうしよ？」と聞かれた時は、`recipes/`（自炊）と `restaurants/`（外食）の両方から候補を出す。体調・気分・時間で絞り込み。
